# 🌱 EcoAgro – Fazenda Sustentável
Bem-vindo ao EcoAgro, um projeto criado para conscientizar sobre a importância de equilibrar a produção agrícola e a preservação do meio ambiente. 🌾🌳

# 🎯 Objetivo
- O jogador assume o papel de um produtor rural e precisa tomar decisões para desenvolver sua fazenda de forma sustentável, mantendo o equilíbrio entre:
- 🌾 Produção agrícola
- 💧 Uso responsável da água
- 🌳 Preservação ambiental
- O desafio é mostrar que é possível produzir alimentos e, ao mesmo tempo, cuidar da natureza.

# 🕹️ Como Funciona
Ao longo do jogo, o jogador poderá escolher diferentes ações, como:
- 💧 Utilizar irrigação inteligente
- 🌳 Plantar árvores
- ☀️ Investir em energia solar
- 🚜 Melhorar a produção da fazenda
- Cada escolha traz consequências positivas ou negativas para os recursos da propriedade.

# 🌎 Relação com o Tema do Agrinho 2026
- O projeto foi desenvolvido com base no tema:
"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente."
- A proposta é demonstrar, de forma interativa e educativa, como a tecnologia e as boas práticas podem contribuir para um futuro mais sustentável no campo.

# 🚀 Tecnologias Utilizadas
- HTML

# 📚 Aprendizados
- Com este projeto, é possível aprender sobre:
- Sustentabilidade no agronegócio 🌱
